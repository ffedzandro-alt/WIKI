# Cópias de Segurança

## Objetivo

Garantir a recuperação do sistema em caso de falha.

## Base de Dados

1. Abrir o phpMyAdmin.
2. Selecionar a base de dados.
3. Exportar em formato SQL.

## Ficheiros

Efetuar cópia da pasta do projeto Joomla.

Inclui:

- templates
- images
- plugins
- configuration.php

## Frequência

- Antes de atualizações
- Semanalmente
- Antes de alterações importantes

## Armazenamento

As cópias devem ser guardadas num local seguro.
