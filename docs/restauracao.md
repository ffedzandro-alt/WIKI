# Reposição de Versão

## Objetivo

Restaurar o sistema utilizando uma cópia de segurança.

## Passos

1. Restaurar todos os ficheiros do Joomla.
2. Importar a base de dados SQL.
3. Atualizar o ficheiro `configuration.php`.
4. Confirmar a ligação à base de dados.
5. Testar todas as funcionalidades.

## Verificações

Após a reposição verificar:

- Login
- Página inicial
- Categorias
- PDFs
- Permissões
- Documentação
- Pesquisa

## Controlo de versões

Caso seja necessário voltar para uma versão anterior, utilizar o histórico disponível no GitHub.
