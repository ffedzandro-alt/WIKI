# Wiki Interna - TechBase

## Descrição

A Wiki Interna é uma plataforma desenvolvida em Joomla 6.1.1 destinada à gestão e consulta de documentação técnica da empresa.

O sistema permite organizar informação por categorias, disponibilizar documentação interna, incorporar ficheiros PDF e gerir permissões de acesso por utilizador.

## Tecnologias utilizadas

- Joomla 6.1.1
- PHP 8
- MySQL
- HTML5
- CSS3
- JavaScript
- Git
- GitHub
- Markdown

## Funcionalidades

- Página inicial personalizada
- Gestão de documentação interna
- Incorporação de PDFs
- Sistema de permissões por utilizador
- Interface responsiva
- Documentação técnica em Markdown
- Controlo de versões com Git
