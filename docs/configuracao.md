# Configuração

## Base de Dados

Configurar os seguintes parâmetros:

- Host
- Nome da Base de Dados
- Utilizador
- Palavra-passe

## Utilizadores

O sistema possui diferentes níveis de acesso.

### Administrador

Pode:

- Gerir utilizadores
- Criar páginas
- Editar conteúdos
- Configurar o Joomla

### Editor

Pode:

- Criar artigos
- Editar artigos

### Leitor

Pode:

- Consultar apenas a documentação autorizada

## Plugins

Ativar:

- PDF Embed
- Logical-Arts Markdown

## Git

O código encontra-se versionado através do Git e alojado no GitHub.
