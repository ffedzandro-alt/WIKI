# Instalação

## Requisitos

- PHP 8 ou superior
- Joomla 6.1.1
- MySQL
- Apache ou Nginx

## Instalação

1. Instalar o servidor web.
2. Criar uma base de dados MySQL.
3. Instalar o Joomla.
4. Configurar o ficheiro `configuration.php`.
5. Instalar os plugins necessários.
6. Importar os conteúdos da Wiki.

## Plugins utilizados

- PDF Embed
- Logical-Arts Markdown

## Testes

Após a instalação verificar:

- Login
- Página inicial
- Categorias
- PDFs incorporados
- Permissões dos utilizadores
